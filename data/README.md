# Descripción datasets utilizados

## originales
Datasets del *Museo del Prado* con detalles de usuarios de tipo demográfico u opinión y datos sobre obras de arte.

## countries.csv
Cada elemento del conjunto de datos es un páis descrito por una serie de características sociales, económicas, geográficas y demográficas. Extraído de *https://www.kaggle.com/noxmoon/world-countries-predicting-gdp*

## countries_distance.csv
Cada elemento del conjunto de datos es un páis descrito por su índice de alfabetismo y PIB per cápita, así como por un factor calculado a partir de ambas para establecer las distancias entre los países.
